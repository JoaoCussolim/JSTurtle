Ajuda - Introdução ao Código

O que são variáveis?

Variáveis são espaços onde você armazena dados temporariamente. Você pode pensar nelas como caixas que contêm valores. Por exemplo:

let x = 5;

Aqui, a variável x armazena o valor 5.

O que são loops?

Loops permitem que você repita um conjunto de instruções várias vezes. Por exemplo, o for é usado para repetir ações de acordo com uma condição.

for (let i = 0; i < 10; i++) {
            console.log(i);
        }

Esse código irá imprimir os números de 0 a 9.

X e Y no plano

O plano cartesiano é composto por dois eixos: X (horizontal) e Y (vertical). Ao utilizar X e Y, você define a posição de objetos no espaço.

drawCircle(10, 10, 20, 'blue');

Neste caso, o círculo será desenhado na posição X = 10 e Y = 10, raio de valor 20 e cor Azul.

Funções disponíveis:

  • Função 1: clear() - Limpa a tela
  • Função 2: drawSquare(x, y, size, color) - Desenha um quadrado
  • Função 3: drawCircle(x, y, size, color) - Desenha um circulo
  • Função 4: drawRectangle(x, y, width, height, color) - Desenha um retângulo
  • Função 5: drawTriangle(x1, y1, x2, y2, x3, y3, color) - Desenha um triângulo
  • Função 6: drawHeart(x, y, size, color) - Desenha um coração
  • Função 7: middleX() - Retorna o valor do X do centro
  • Função 8: middleY() - Retorna o valor do Y do centro
